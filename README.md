# Cài đặt

### Yêu cầu 

* Node: version >= 13
* yarn

Bước 1:
`git clone --depth=1 https://github.com/HaiDang309/api-nhom16.git <YOUR_FOLDER>`

Bước 2:
`cd <YOUR_FOLDER>`

Bước 3:
`yarn` hoặc `yarn install`

Bước 4:
`cp .env.example .env`

Bước 5:
`yarn dev`

## Lưu ý:
- Chỉnh sửa thông tin trong file `.env` cho phù hợp
